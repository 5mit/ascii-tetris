
# **ascii-tetris**

### A CLI version of Tetris implementated with ncurses

------------

## Compilation

Type `make` in the project directory; make sure the host system has ncurses installed. (Most Linux environments do)

## How To Run

Type `./tetris` in the project directory.

## Controls

w: rotate piece\
a: move piece left\
s: move piece down\
d: move piece right
